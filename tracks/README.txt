put your own audio files here and the game plays them natively
(no youtube iframe = no lag). any track that's missing simply falls
back to the hidden youtube player, so you can fill this in gradually.

file names the game looks for (mp3):

  happysad.mp3      — happysad
  kickinit.mp3      — kickinit alone
  bridges.mp3       — bridges
  filesystem.mp3    — kiefer's file system
  superbloom.mp3    — superbloom
  betweendays.mp3   — between days
  lovearound.mp3    — when there's love around
  itsokbu.mp3       — it's ok, b u
  fathers.mp3       — fathers
  club.mp3          — the secret gig (live at wonderland park)

where to get the files legally: kiefer's albums are on bandcamp
(kiefer.bandcamp.com) — buying an album gives you DRM-free mp3
downloads you can drop straight in here.

note: this folder is git-ignored on purpose. do NOT commit purchased
music to the public github repo — that would be redistributing it.
